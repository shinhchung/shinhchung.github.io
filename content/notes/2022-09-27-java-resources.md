I just applied for Amazon Technical Academy. It requires learning Java. I know Java, but not very deeply. Here are resources an Amazon cohort shared with me.

## Books and Blog Posts

- [Head First Java (free version)](https://www.rcsdk12.org/cms/lib/NY01001156/Centricity/Domain/4951/Head_First_Java_Second_Edition.pdf)

## Practice and Challenges

- [CodingBat Java](https://codingbat.com/java) — Various practice exercises
- [Advent of Code](https://adventofcode.com/) — A series of puzzles for problem solving and coding
- Online practice tests for Java Certification:
  - http://www.gocertify.com/quizzes/oracle/ocajp-java-quiz-1.html
  - https://www.javatpoint.com/core-java-quiz
  - https://www.examsnet.com/exams/ocjp-certification-practice-online

## Talks

- [A humorous talk on silly programming language behavior](https://www.destroyallsoftware.com/talks/wat)
